# PerpDex MVP Demo Checklist

## 1. 钱包连接与登录
- [ ] 连接 MetaMask 钱包
  - 验证：钱包地址正确显示
- [ ] SIWE 签名登录
  - 验证：签名请求弹出，登录成功
- [ ] 登录状态持久化
  - 验证：刷新页面后仍保持登录状态

## 2. 充值流程
- [ ] 查看当前余额
  - 验证：显示 Available Balance 和 Total Balance
- [ ] USDC Approve 授权
  - 验证：MetaMask 弹出授权交易
- [ ] Deposit 交易确认
  - 验证：交易成功后余额增加
- [ ] 余额实时更新
  - 验证：无需手动刷新页面

## 3. 交易流程
- [ ] 查看当前价格
  - 验证：显示 BTC/USDC 实时价格
- [ ] 开多仓位（LONG）
  - 验证：仓位创建成功，保证金锁定
- [ ] 开空仓位（SHORT）
  - 验证：仓位创建成功，保证金锁定
- [ ] PnL 实时显示
  - 验证：盈亏随价格变动实时更新
- [ ] 风险等级指示
  - 验证：SAFE/WARNING/DANGER 状态显示

## 4. 平仓流程
- [ ] 手动平仓
  - 验证：平仓交易成功
- [ ] 保证金释放
  - 验证：Available Balance 增加
- [ ] PnL 结算
  - 验证：盈亏正确计入余额
- [ ] 仓位状态更新
  - 验证：仓位列表中不再显示已平仓

## 5. 提现流程
- [ ] 发起提现请求
  - 验证：输入金额验证
- [ ] 提现交易确认
  - 验证：MetaMask 弹出交易
- [ ] 余额扣减
  - 验证：提现后余额正确

## 6. 异常流程
- [ ] 余额不足时开仓
  - 验证：显示错误提示
- [ ] 重复开仓（同币种同方向）
  - 验证：显示"仓位已存在"错误
- [ ] 网络错误处理
  - 验证：显示友好的错误消息
- [ ] 交易超时处理
  - 验证：超时后显示重试选项

## 7. 性能验证
- [ ] 页面加载时间 < 3s
- [ ] 交易响应时间 < 2s
- [ ] WebSocket 连接稳定

## 8. 安全验证
- [ ] 未登录无法访问交易功能
- [ ] JWT Token 正确过期
- [ ] 敏感操作需要重新验证
